Monitoring

Error: Cannot find Mongo Collection

This error can occur because this app does not have an upgraded Mongo Database.

This can be corrected in your App Studio with the following steps:

  1. Click into this app's data browser (for the current environment)
  2. click 'upgrade database' to upgrade the collection to dedicated collection
  3. click 'Upgrade now'.
  4. Click the 'deploy' button in the prompt that shows up after upgrading.
  5. Click 'deploy cloud app'

Once that has completed, this issue should be resolved.
